# crwn clothing Project

E-commerce platform for clothes and footwear.

Technologies used:

- React (Create React App)
- React Router
- React Contex
- Redux
- Redux Persist
- Redux Thunk
- Redux Saga
- Reselect (memoization)
- Firebase (authentication and database)
- Stripe (payments)
- Typescript
- GraphQL + Apollo
- SASS
- Media queries
